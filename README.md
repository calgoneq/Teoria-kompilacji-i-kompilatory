# Teoria-kompilacji-i-kompilatory


TEMAT PROJEKTU: 
Kompilator autorskiego, proceduralnego języka opisu grafiki wektorowej do formatu SVG.

OPIS:
Chcielibyśmy napisać kompilator własnego języka do generowania grafiki. 
Kod źródłowy będzie przetwarzany na AST (drzewo składniowe), a następnie tłumaczony na format XML/SVG. 
Język będzie obsługiwał matematykę, pętle oraz definiowanie i wywoływanie własnych procedur z parametrami. 
Użyjemy do tego generatora parserów SLY w Pythonie. Planujemy tez zaimplementować obsługę błędów
